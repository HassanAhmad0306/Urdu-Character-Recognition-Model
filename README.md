# Urdu Character Recognition using CNN

## Overview
This project uses a Convolutional Neural Network (CNN) to classify Urdu characters from images.

## Technologies
- Python
- TensorFlow/Keras
- Google Colab
- CNN

## Features
- Image preprocessing
- CNN model training
- Urdu character classification
- Accuracy evaluation

## Future Improvements
- Real-time Urdu handwriting recognition
- Larger dataset support
